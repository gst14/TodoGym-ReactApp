> Una Web para gimnasios construida con Mongo, Express, Node and React (MERN).

## Clone

```sh
git clone "URL"
```

## Install

```sh
npm i
```
Back y Frontend (cd /client)

## Usage

```sh
npm run dev

cd /client
npm start
```

## Run tests

```sh
npm run test
```

Se debe hacer tanto en front como en Back
